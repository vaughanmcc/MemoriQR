# Marketing Materials

This folder contains downloadable marketing materials for MemoriQR partners.

## Structure

```
materials/
├── memoriqr-brochure.pdf         # General brochure
├── memoriqr-pet-brochure.pdf     # Pet memorial brochure
├── memoriqr-human-brochure.pdf   # Human memorial brochure
├── memoriqr-price-list.pdf       # Retail price list
├── logos/
│   ├── memoriqr-logo-color.png   # Full color logo (transparent)
│   ├── memoriqr-logo-white.png   # White logo (transparent)
│   ├── memoriqr-logo-black.png   # Black logo (transparent)
│   └── memoriqr-logo-pack.zip    # All formats (PNG, SVG, EPS)
├── displays/
│   ├── counter-display-a5.pdf    # A5 counter card
│   ├── poster-a3.pdf             # A3 poster
│   └── window-decal.pdf          # Partner window sticker
└── social/
    ├── social-media-pack.zip     # FB, IG, LinkedIn graphics
    └── canva-templates.pdf       # Links to editable Canva templates
```

## Adding Files

1. Create the actual PDF/image files
2. Place them in the appropriate subfolder
3. File names must match exactly as listed above

## File Specifications

### Brochures
- Format: PDF
- Size: A4 portrait or landscape
- Resolution: 300 DPI for print

### Logos
- PNG: Transparent background, min 1000px width
- SVG: Vector format for scaling
- EPS: For professional print

### Display Materials
- Print-ready PDF with bleed marks
- CMYK color space

### Social Media
- Facebook: 1200x630px
- Instagram: 1080x1080px (square), 1080x1920px (stories)
- LinkedIn: 1200x627px
